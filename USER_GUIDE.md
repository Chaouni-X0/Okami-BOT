## 1. التثبيت والإعداد الأولي
يتطلب البوت بيئة عمل **Node.js (v18+)**.

### خطوات التثبيت:
1. قم بتحميل ملفات المشروع أو استنساخ المستودع من GitHub.
2. افتح المجلد في Terminal وقم بتثبيت المكتبات:
   ```bash
   pnpm install
   ```
3. قم بإنشاء ملف `.env` في المجلد الرئيسي وضع فيه الإعدادات التالية:
   ```env
   FB_PAGE_TOKEN=your_page_access_token
   FB_PAGE_ID=your_page_id
   ADMIN_KEY=chaouni_x_2013-2
   PORT=3000
   ```

## 2. ضبط إعدادات فيسبوك
للحصول على أفضل أداء ومنع الحظر:
- استخدم **Page Access Token** طويل الأمد (Long-lived Token).
- تأكد من منح البوت صلاحيات: `pages_manage_posts`, `pages_read_engagement`, `pages_show_list`.
- **نظام الحماية (Anti-Ban):** البوت مبرمج تلقائياً لعمل تأخير عشوائي بين المنشورات (30-120 ثانية) لمنع خوارزميات فيسبوك من رصده كسبام.

---

## 3. نظام التفاعل (Gamification)
Okami Bot يعتمد على تحويل القراءة إلى لعبة:
- **XP & Levels:** كلما قرأ المستخدم فصولاً أكثر، زاد مستواه.
- **Ranks:** يبدأ المستخدم كـ `Otaku Beginner` ويصل إلى `Okami King 🐺`.
- **Daily Streak:** الحضور اليومي يمنح نقاطاً مضاعفة. انقطاع الحضور يصفر الـ Streak ما لم يستخدم المستخدم "Streak Freeze".

---

## 4. نظام القبائل والفعاليات
### القبائل (Guilds):
يمكنك إنشاء قبائل (مثل: قبيلة الظل، قبيلة التنين) والسماح للمستخدمين بالانضمام إليها. نقاط الأفراد ترفع ترتيب القبيلة.

### الفعاليات (Events):
يمكنك إطلاق فعاليات مؤقتة مثل "نهاية أسبوع النقاط المضاعفة". خلال هذه الفترة، كل تفاعل يمنح نقاطاً أكثر بمرتين أو ثلاث.

---

## 5. وضع المحادثة التفاعلي (Interactive Chat Mode) 🐺💬
يمكنك الآن التحدث مع البوت مباشرة عبر Messenger لاستخدامه كـ "مساعد ذكي":

### 🛠️ خطوات استخدام وضع المطور التفاعلي:
1. أرسل **"مرحبا"** للبوت.
2. اختر **"وضع المطور"** (رقم 1).
3. أدخل **كلمة السر** (`chaouni_x_2013-2`).
4. اختر **الموقع** من القائمة المرقمة (1-8).
5. أرسل **Slug** العمل (مثال: `solo-leveling`).
6. سيبدأ البوت فوراً بالنشر وسيرسل لك رسالة تأكيد عند الانتهاء.

### 📖 وضع القارئ التفاعلي:
يمكن للمستخدمين اختيار رقم 2 لاستعراض ملفاتهم الشخصية، الـ Streaks، والمهام اليومية مباشرة من المحادثة.

## 6. لوحة تحكم المطور (Admin API)
بجانب وضع المحادثة، يمكنك استخدام الـ API لطلبات `POST` المتقدمة:

### الأوامر الأساسية:
| الأمر | الوصف | البارامترات المطلوبة |
| :--- | :--- | :--- |
| `QUICK_ADD` | إضافة مانهوا ونشر آخر فصل فوراً | `source`, `slug` |
| `GET_SOURCES` | عرض المواقع الـ 8 المدعومة | لا يوجد |
| `START_EVENT` | بدء فعالية فورية | `name`, `type`, `hours` |
| `REWARD_GUILD` | مكافأة قبيلة كاملة | `guildId`, `amount` |
| `WARN_USER` | تحذير مستخدم (3 = حظر) | `fbId`, `reason` |

### 🌍 المواقع المدعومة (8+):
يدعم البوت حالياً: **Azora Moon, Swat Manga, Team X, Manga Arab, Manga Lek, Ares Manga, Galaxy Manga, G-Manga**.
يمكنك إضافة أي عمل من هذه المواقع باستخدام `QUICK_ADD`.

## 6. واجهة المستخدم (User API)
هذه الواجهة مخصصة لربط البوت بتطبيق أو موقع أو حتى استخدامه عبر Messenger:
- `GET /user/profile/:fbId`: لعرض بطاقة المستخدم الإحصائية.
- `GET /user/missions/:fbId`: لعرض المهام اليومية المطلوبة.
- `POST /user/read`: لتسجيل قراءة فصل وتحديث التقدم.

## 7. أفضل الممارسات للنمو
1. **استخدم الـ Broadcast بذكاء:** لا ترسل إعلانات كثيرة، استخدمها فقط للفعاليات الكبرى.
2. **شجع التنافس بين القبائل:** انشر قائمة المتصدرين أسبوعياً وكافئ القبيلة الفائزة.
3. **المهام اليومية:** تأكد من أن المهام بسيطة وممتعة لتشجيع المستخدمين على العودة.
4. **تفاعل البوت:** استخدم نظام الردود الذكية لجعل البوت يبدو "بشرياً" في تعليقات الصفحة.
