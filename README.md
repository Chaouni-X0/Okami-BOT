# 🐺 Okami Bot - بوت أوكامي v3.0

> بوت بايثون متقدم لنشر المانغا والمانهوا من المواقع العربية على فيسبوك تلقائياً

## ✨ الميزات الرئيسية

- نشر آلي من 7+ مواقع عربية مشهورة
- تقسيم ذكي للصور الطويلة بأعلى جودة
- علامة مائية تلقائية على كل صورة
- نظام إشعارات ومتابعة متقدم
- نظام تقييم ومراجعات
- قائمة "اقرأ لاحقاً"
- توصيات ذكية مخصصة
- إحصائيات قراءة شخصية
- حذف تلقائي لتوفير المساحة
- جدولة تلقائية للفصول الأسبوعية
- متوافق مع Replit (24/7)
- أوامر مطور وأوامر مستخدم منفصلة

## 🚀 التشغيل

```bash
pip install -r requirements.txt
# عدّل config.env بإضافة التوكن ومعرف الصفحة
python main.py
```

## 🔐 كلمة سر المطور: أدخل `chaouni_x_2013-2` لفتح أوامر الإدارة

## 📋 الأوامر: أدخل `help` لعرض جميع الأوامر المتاحة
